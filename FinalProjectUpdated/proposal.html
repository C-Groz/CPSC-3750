Chosen Base App

I selected the Simple Calendar App from the textbook. It includes features such as displaying the dates for any month between 1990 and 2020 and adding/viewing custom events.

Proposed Project Overview

My application will allow users to register for an account and login to:

  • Add custom events to the calendar and view them ordered by time/date
  • Edit events
  • Create categories of events
  • Create reoccuring events
  • View the calendar based on years, months, or weeks
  • Search events
  • Upload files to events (eg. photos/text files)
  • View a user profile page which shows:
    • User information
    • Upcoming events
    • User statistics

Additionally, it will contain an admin dashboard which:

  • Shows information on all users of the system
  • Allows for deletion of users

Enhancements

In order to acheive the proposed project above, I will enhance the following:

  • HTML: Improved layout, create additional pages(admin dashboard, user profile)
  • CSS: Responsive design, dark mode toggle, consistent styling, enhanced UI for admin dashboard/user profiles
  • JavaScript: Form validation, AJAX for searching and updating calendar w/o refresh, new calendar views (day, month, year)
  • PHP: Develop functionality for editing, searching, viewing, and deleting events, add functionality for reocurring events, add functionality for admin dashboard and user profiles, update to use dates past 2020
  • Database: Add search functionality, allow file uploads, deletion of users, queries for user stats, add column for user role
  • Authentication: Login/Logout with password hashing, role creation (user/admin)
